## About The Project

Features:
* Highly customizable discord messages for each event
* Posibility to send each meassge to a different channel
* Player count on bot status
* kick, ban players with bot commands (coming soon)

<!-- GETTING STARTED -->
## Getting Started

These are the steps to get the bot up and running on your server with basic configuration.

### Prerequisites

These are the prerequisites to run the bot on your server.

* Discord Bot Token (https://discord.com/developers/applications)
* Windows or Windows Server Operating System
* x64 (64-bit) architecture (you probably have this one)
* .NET Runtime 8.0 or higher (https://dotnet.microsoft.com/download)

Nothing else is required to run the bot on your server.

### Bot Permissions

The bot needs the following permissions to work properly:
* Send Messages
* Embed Links
* View Channels

<!-- USAGE EXAMPLES -->
## Usage

These are the steps to install and run the bot on your server.

1. Download the latest release zip file from the releases page [Releases](https://github.com/GreenLeaf-Gaming/MotorTownDiscordBot/releases/latest)
2. Extract the zip file to a folder on your server
3. Copy or rename the `config.sample.json` file to `config.json`
4. Edit the `config.json` file with your bot token and channel settings
5. Run the `MotorTownDiscordBot.exe` file

_For more examples, please refer to the [Wiki](https://github.com/GreenLeaf-Gaming/MotorTownDiscordBot/wiki)_
